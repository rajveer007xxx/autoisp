Equal split. Sticky per connection so streams don't break. Most common ISP dual-WAN setup.

PCC Weighted

Custom % per WAN — useful when one link is faster or has different cost.

Failover

WAN1 carries 100%. WAN2 takes over only when WAN1's gateway stops responding to ping.
