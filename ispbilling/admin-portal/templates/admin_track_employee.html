About GPS Tracking

Mobile App Integration

This GPS tracking feature is designed to work with the mobile app. When employees use the mobile app, their location data will be automatically captured and displayed here in real-time.

Features

  • Real-time GPS location tracking
  • Location history with timestamps

  • Route visualization on map
  • Geofencing alerts (coming soon)

Privacy & Permissions

Location tracking requires employee consent and is only active when the mobile app is in use. Employees can view their own location history and tracking status through the mobile app.
